Read Me:
This is a MVC Twitter-a-like used to handle Events (Via a hashtag) and create stories based upon them. It uses AngularJS and Ajax to be as dynamic as possible without needing to reload etc.

as such the  application relies on a RESTful api.

:::::HOW TO USE:::::

**FIRST do a composer install on the root of the folder. 
**Then Create a database named mystories in MSQL in PHP myAdmin (The process for creating a database in laravel is quite indepth, it is much easier this way).
**Then run php artisan migrate
**Then run php artisan db:seed
**Then run php artisan serve


Default User is TestAct with password test. 

The application will then be running on localhost:8000.
